## Principles

1. **Write in plain markdown** - No proprietary formats
2. **Link freely** - Create connections between ideas
3. **Review regularly** - Notes without review are worthless
4. **Keep it simple** - Resist over-organization
5. **Future-proof** - Plain text lasts forever
