# Safi Development

**Status:** Active
**Started:** October 2025
**Tech Stack:** Nuxt 4, Vue 3, TypeScript, TailwindCSS, CodeMirror 6

## Overview
A minimalist, self-hosted markdown editor focused on simplicity and mobile experience. Built with modern web technologies and designed for personal knowledge management.

## Goals
- Create the simplest possible markdown editing experience
- Ensure flawless mobile/touch interactions
- Maintain zero-dependency server-side storage (just files)
- Keep the codebase clean and maintainable

## Technical Decisions
### Why Nuxt 4?
- SSR capabilities for future features
- Great DX with Vue 3
- Built-in API routes via Nitro

### Why File-Based Storage?
- Simplicity - no database needed
- Portability - users own their data
- Git-friendly - version control works naturally
